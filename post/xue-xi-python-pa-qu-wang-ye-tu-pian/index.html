学习python爬取网页图片

2020-03-27

一、需要安装requests
1、Windows键+R，回车。输入cmd

2、输入pip install requests

二、程序
import requests
pic = requests.get('https://p.ssl.qhimg.com/dmsmty/74_74_100/t0117cd78b141e0bfa2.png')#在你需要的爬取的网页F12、ctrl+shift+c、点击你需要的图片找到src=“”里面的
with open('python.png','wb') as f:#保存的文件名
f.write(pic.content)

三、就会生成一个.png的文件
